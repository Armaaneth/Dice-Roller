Dice Roller
This is a dice roller app that was developed using Android and Kotlin. It is a simple app that allows users to roll a dice and see the result.

Features
The app allows users to roll a dice of any number of sides.
The app shows the result of the dice roll in a text box.
The app has a button that allows users to roll the dice again.
How to Use
To use the app, simply open it and tap on the "Roll Dice" button. The app will then roll a dice and show the result in a text box. To roll the dice again, simply tap on the "Roll Dice" button again.
